# AI Resume Screening & Job Matching System

AI-powered resume analyzer using NLP and Machine Learning.

## Features

- Resume PDF Upload
- Skill Extraction
- Job Recommendation
- Match Percentage
- Missing Skills Detection

## Technologies

- Python
- Streamlit
- Scikit-learn
- NLP
- TF-IDF
- Cosine Similarity

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
